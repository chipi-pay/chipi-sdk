# createWallet(pin)

Creates a new Argent-compatible wallet on StarkNet. This function uses Avnuâ€™s paymaster to sponsor gas fees, making the wallet creation process completely frictionless for end users.

## Parameters

- **pin** (string): A user-defined numeric code or password used to encrypt the generated private key.

## Usage

```ts
import { ChipiSDK } from '@chipi-pay/chipi-sdk';

const sdk = new ChipiSDK({
  // Required fields
  apiKey: 'your-paymaster-api-key', 
  rpcUrl: 'https://starknet-goerli.infura.io/v3/your-api-key', 
  // Optional fields
  argentClassHash: '0x...',
  contractAddress: '0x...',
  contractEntryPoint: 'get_counter',
});

async function createArgentWallet() {
  const result = await sdk.createWallet('1234');
  console.log(result);
  // Returns: { success: true, txHash: '0x...'}
}
```

## Return Value

This function returns a `Promise<TransactionResult>` object:

- **success** (boolean): Indicates whether the wallet creation transaction was executed successfully.
- **txHash** (string): The resulting transaction hash on StarkNet.

## Example

If you receive `{ success: true }`, the Argent wallet contract has been deployed. You can then retrieve the wallet details (e.g., public key, encrypted private key) from your configured storage or follow up with additional logic, such as prompting the user to fund their wallet or proceed with other dApp interactions.
