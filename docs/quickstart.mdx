## Getting Started

Before you begin, make sure you've completed the steps in our [Prerequisites](/prerequisites) page. This ensures you have the SDK, RPC, and paymaster keys set up.

Kick off your journey with our **Chipi SDK** for setting up invisible wallets on StarkNet using Avnu's paymaster. Below is a quick example of how to integrate the SDK and perform basic wallet operations.

<CodeGroup>

```js React
import { ChipiSDK } from '@chipi-pay/chipi-sdk';

// Initialize SDK
const sdk = new ChipiSDK({
  paymasterApiKey: 'your-api-key',
  rpcUrl: 'https://starknet-goerli.infura.io/v3/your-api-key',
  argentClassHash: '0x...', // optional
  contractAddress: '0x...',  // optional
  contractEntryPoint: 'get_counter' // optional
});

// Create wallet
const creationResult = await sdk.createWallet('1234');
// Returns: { success: true, txHash: '0x...' }

// Execute a transaction
const txHash = await sdk.executeTransaction({
  pin: '1234',
  wallet: {
    publicKey: '0x...',
    encryptedPrivateKey: 'encrypted...',
  },
  calls: [
    {
      contractAddress: '0x...',
      entrypoint: 'increment',
      calldata: ['0x...']
    }
  ]
});
// Returns: string (transaction hash) or null if error
```
</CodeGroup>
